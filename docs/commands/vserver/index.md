# vServer Commands

vServer (VNG Virtual Server) commands for managing virtual machine instances.

```bash
grn vserver <command> [options]
```

## Available commands

### Server

| Command | Description |
|---------|-------------|
| [list-servers](list-servers.md) | List all vServer instances |
| [get-server](get-server.md) | Get vServer instance details |
| [create-server](create-server.md) | Create a new vServer instance |
| [start-server](start-server.md) | Start a stopped vServer instance |
| [list-flavors](list-flavors.md) | List available flavors by zone and instance family |
| [stop-server](stop-server.md) | Stop a running vServer instance |
| [reboot-server](reboot-server.md) | Reboot a vServer instance |
| [delete-server](delete-server.md) | Delete a vServer instance |

### Volume

| Command | Description |
|---------|-------------|
| [resize-volume](resize-volume.md) | Resize a volume's size or change its volume type |
