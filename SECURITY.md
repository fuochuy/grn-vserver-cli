# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the GreenNode CLI, please report it by email to **security@vngcloud.vn** rather than opening a public GitHub issue.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix within 90 days of disclosure. You will be credited in the release notes unless you request anonymity.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | Security fixes only |

## Scope

This policy covers the `grn` CLI binary and this repository. It does **not** cover:

- The VNG Cloud API — report API issues to [VNG Cloud Support](https://support.vngcloud.vn)
- Third-party dependencies — report those upstream

## Disclosure Policy

We follow coordinated disclosure. We ask that you allow us reasonable time to remediate before public disclosure.
