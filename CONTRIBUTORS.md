## How to get listed

Contributions of any kind are welcome — code, documentation, bug reports, and feature requests.

Open a pull request or issue at [github.com/vngcloud/greennode-cli](https://github.com/vngcloud/greennode-cli).
All merged PR authors are automatically credited in the release notes.

See [CONTRIBUTING.md](CONTRIBUTING.md) for the full contribution guide.
